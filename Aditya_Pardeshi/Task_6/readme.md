## Team 3 – AI Fake News Detection System 
Objective: 
Build a text classification model to detect fake vs real news. 
Requirements: 
• Data preprocessing 
• TF-IDF / Word Embeddings 
• Train multiple models (Naive Bayes, Logistic Regression, etc.) 
• Compare model accuracy 
• Confusion matrix & evaluation 
• Deploy mini web app 
Difficulty: Medium to Advanced NLP
