# Task 3: Digit Recognition Web App (Streamlit)

## Objective
Deploy a CNN model trained on MNIST dataset to predict handwritten digits.

## Features
- Upload digit image  
- Predict digit using trained CNN model  
- Display prediction and confidence  

## How to Run

Open in browser: http://localhost:8501

## Technologies Used
Python, TensorFlow, Streamlit, OpenCV, NumPy
