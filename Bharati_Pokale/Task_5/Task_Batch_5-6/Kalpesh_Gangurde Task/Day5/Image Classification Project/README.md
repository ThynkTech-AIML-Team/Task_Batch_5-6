# Image Classification Project (MNIST)

## Files
- `image_classification_mnist.ipynb`: Complete runnable notebook.

## Requirements
Install packages:
```bash
pip install numpy matplotlib seaborn scikit-learn tensorflow
```

## Run
1. Open project folder in VS Code.
2. Open `image_classification_mnist.ipynb`.
3. Select a Python kernel.
4. Run all cells.

## Output
- Trains Linear SVM and CNN on MNIST
- Prints test accuracy for both models
- Shows confusion matrices
- Shows CNN training vs validation accuracy plot
- Prints final performance comparison
