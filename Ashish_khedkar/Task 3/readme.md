
# README

## 1. Text Classification (Spam Detection)

**Dataset:** SMS Spam Collection (spam / ham messages)
**Models:** Naive Bayes, Logistic Regression, SVM
**Features:** CountVectorizer, TF-IDF
**Result:** Countvectoriser + Logistic Regression/SVM gave the best accuracy.

---

## 2. Word Embedding Task

**Embeddings:** Word2Vec (Google News), GloVe (Wikipedia)
**Tasks:** Similar words, word analogy, PCA visualization
**Result:** Word2Vec performed better for analogies; GloVe was consistent for similarity.

---

## 3. Fake News Detection

**Dataset:** Fake vs Real News articles
**Models:** Logistic Regression, Naive Bayes
**Features:** TF-IDF
**Result:** Logistic Regression achieved higher accuracy of 91%.
